Hotel and restaurant reservation

== Description ==
This is a simple plugin where you can create a list of hotel/restaurant and book them with basic booking system.

= Features =

* 	Hotel Post Type post
*	Extra hotel information field in posts
*       Three display view
*       Extra Gallery for hotel
*       Shortcode Feature with parameter
*       Plugin Template Support
*       Single Page Template
*       Basic Hotel Global Setting

== Installation ==

1. Uncompress the download package
1. Upload folder including all files the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
